\# Banking Management System



A Python Banking System built using SQLite and bcrypt.



Features:

\- User Registration

\- Secure Login

\- Password Hashing

\- Deposit Money

\- Withdraw Money

\- Transfer Money

\- Balance Check

\- Transaction History

\- Mini Statement

\- Change Username

\- Change Password

\- Change PIN

\- Account Details

\- Admin Panel

\- View Accounts

\- View Users

\- Total Bank Balance

\- View All Transactions

\- Search Account



Tech Stack:

\- Python 3.11

\- SQLite3

\- bcrypt



Run:

python main.py
